## 설치
```bash
npm install
```

## 실행
```bash
npm start
```

## 빌드
```bash
npm run build
```

## 사용 기술
- React + TypeScript
- Recoil 상태 관리
- React Router DOM
- Sass 스타일링

## 개발 환경
- Node.js 18 이상
- Create React App(TypeScript)
